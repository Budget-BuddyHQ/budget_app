# Project: Budget Buddy (Prodigy-Style RPG)
## Core Vibe
- 2.5D Isometric World Map.
- Mascot: Merchant Turtle.
- Goal: Build a kingdom using "Shell-coins" earned from financial lessons.

## Technical Standards
- Framework: Flutter.
- State Management: Provider/ValueNotifier.
- Required Mixin: TickerProviderStateMixin (for multi-animation support).
- Asset Type: SVG/Vector for high-performance mobile scaling.
